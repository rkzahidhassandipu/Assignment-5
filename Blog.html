Question-1: What are the different ways to select an element in the DOM?

DOM (Document Object Model), I use GetElementByID to select an element by an ID. Select an element by a class name with GetElementByClassName. Select an HTML tag using getElementByTagName. CSS Class Name, ID name, HTML tag is a selector used in a querySelector. If I have a class name, HTML Tag, ID Name, repeated in my HTML page Then I want to select them in order, in that case I use the queryselectorall.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML

If I want to create a tag inside the HTML with the JavaScript, I have to use innerHTML and it can also set the stream text but it should not be set for security reasons (to prevent XSS attacks)

innerText

If I write any HTML tag inside innerText, it will be treated as text. If I select an element and there is an element hidden inside that element, it ignores.

textContent

Explains everything about an element, including the hidden text inside it. It is much like textContent but faster than textContent .

Question-4: What is event bubbling in the DOM?

DOM (Document Object Model) is where an event starts with a target and then propagates (bubbles) to its parent elements and reaches the document object.
